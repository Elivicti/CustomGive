自定义的"/give"指令
====================
在Minecraft中，物品有各种各样的NBT标签，能让"/give"指令能够创造各种神奇的物品，但是由于指令的局限性，导致输入过程过于反人类（无论多长的指令只有一行，还有那无尽的大括号...）<br>
于是我脑洞大开，就有了这个软件，图形界面，使用起来非常亲民，不懂指令也是可以用的<br>

------------------------------------

<br>
